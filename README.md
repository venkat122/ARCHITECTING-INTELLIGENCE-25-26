# ARCHITECTING-INTELLIGENCE-25-26
Official Repository for the Winter Project "Architecting Intelligence" offered by MATSOC 

*PROJECT OBJECTIVE*: This project aims to introduce participants to the core foundations of modern AI systems by covering NLP basics, neural networks, transformer architectures, efficient fine-tuning methods, retrieval systems, and multimodal reasoning.
